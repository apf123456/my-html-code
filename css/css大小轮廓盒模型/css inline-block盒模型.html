1 block此元素将显示为块级元素，此元素默认前后各有换行符。这种盒模型默认占据一行，允许通过css设置宽度，高度，例如div，p，h1到h6，ol，ul，dl，table，blockquote，form 2 inlin此此元素会被显示为内联元素，元素前后没有换行符。这种盒模型元素不会占据一行，默认允许一行放置多个元素，即使css设置高度和宽度也不起作用。例如span，a，br，em，strong，label，q，var，cite，code 3 inline-block 是前两者的综合体，行内块元素。不会默认占据一行，并且可以设置宽度和高度，这样设置可以使得多个div元素并列显示，可以实现多栏布局，默认采用底端对齐模式。
